Solving the pairity 3bit problem using NN with 1 hidden layer.

files:
-CorrectAnswers.m   the correct answers for the problem.

-DeepLearning   where the "Magic" happends.

-ForwardPropagation  forward propagation of the input.

-Get_inputs  get possible inputs (only works for 3bit at the moment).

-Main  where the user inputs a bit array to see it's pairity bit.

-Sigmoid  the activation function.

-Train  the overall training function.
